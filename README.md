# Valentine Card

A cozy "Will you be my Valentine?" single-page card built with Nuxt 3 and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- **Name**: Edit `valentineName` in `pages/index.vue`
- **Colors**: Edit the `valentine` palette in `tailwind.config.ts`
